# HikaShop EwayRapid

eWay Rapid payment plugin for HikaShop (open source e-commerce solution for Joomla).

* PHP 5.4 or newer
* Joomla! 1.5 / 2.5 / 3.x
* HikaShop 2.5.0 or newer
